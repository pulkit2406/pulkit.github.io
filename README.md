# Pulkit Srivastava - Freelance Portfolio Website

This is a GitHub Pages-ready portfolio website for freelancing, Mercor, Upwork, and remote data/AI projects.

## Files included

- `index.html` - Website content and structure
- `style.css` - Responsive styling
- `script.js` - Mobile navigation and footer year
- `assets/Pulkit_Srivastava_Mercor_Freelance_Resume.pdf` - Resume PDF linked from the website

## How to upload to GitHub Pages

1. Open your GitHub repository.
2. Upload or replace these files in the root folder of the repository.
3. Keep the resume PDF inside the `assets` folder.
4. Commit the changes.
5. Open your GitHub Pages URL after the deployment refreshes.
